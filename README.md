# Dialogos Forge

A systematic, human-AI co-learning engine.

## The Spirit: Dialogos
Learning is not a one-way street of information retrieval (prompting an LLM for a summary). True learning requires **Dialogos**—a shared sense-making process where the conversation itself generates insights that neither the human nor the AI could have reached alone. It is tethered climbing: finding footholds in abstract concepts and pulling each other up.

## The Mechanics: The F.O.R.G.E. Protocol
To prevent Dialogos from becoming aimless philosophizing, we run it through a mechanical engine. Every learning loop follows the F.O.R.G.E. protocol:

1. **F - Find (Catalyst)**
   Drop an artifact (a paper, a thought, a code snippet) into the forge. It must be an anomaly or a spark of curiosity.
2. **O - Oppose (Friction)**
   Do not accept simple summaries. Argue with the context. Push back. Translate the abstract into the concrete by exposing the concept to heat and tension.
3. **R - Refine (Synthesis)**
   Compress the chaos of the friction into a single, crystalline blueprint or insight.
4. **G - Ground (Application & Meaning)**
   Ideas floating in the air have no inherent meaning. Grounding forces the idea to touch lived experience by answering: *"How does this change my relationship to the world?"*
5. **E - Embody (Execution)**
   Live the insight. Write the code, log the mental model in a persistent Knowledge Graph, or integrate the pattern into a daily workflow.
