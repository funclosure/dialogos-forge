# Intake Template

Use this when capturing a new incoming learning object before it has fully become a topic thread.

```md
# <Title>

## Type
Dialogos-intake

## Date
YYYY-MM-DD

## Intake Mode
source-digestion | co-reading | skill-acquisition | figure-inquiry | concept-inquiry | embodiment-inquiry

## Themes
comma, separated, themes

## Source
conversation | youtube | podcast | paper | pdf | book | article | code | other

## Why It Has Heat
Why does this feel alive, important, confusing, or charged right now?

## Raw Capture
Links, quotes, fragments, observations, or rough notes.

## Likely Direction
skill path | figure thread | concept thread | synthesis candidate | not sure yet

## First Questions
- 
- 
- 

## Suggested Next Move
What should happen next?

## Linked State
../state/<topic>.md or leave blank if no state file exists yet
```

## Notes

- Use this before over-organizing.
- Not every intake deserves a full topic thread.
- Create a state file only when the thread has enough continuity or energy.
