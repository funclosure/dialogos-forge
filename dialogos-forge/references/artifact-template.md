# Artifact Template

Use this for files in `dialogos/inbox/`, `dialogos/forge/`, or `dialogos/syntheses/`.

```md
# <Title>

## Type
Dialogos-inbox | Forge-output | Synthesis | Memory-candidate

## Date
YYYY-MM-DD

## Themes
comma, separated, themes

## Source
conversation | article | paper | podcast | book | code | other

## Status
raw | in-progress | distilled | embodied

## Prompt or Catalyst
What sparked this thread?

## Body
Main notes, distinctions, arguments, excerpts, or synthesis.

## Linked State
../state/<topic>.md

## Next Step
What should happen next?
```

## Notes

- Use concept-rich filenames, for example `grounding-sean-kelly.md` instead of only `2026-04-21.md`.
- Keep this lightweight.
- If the file is still rough, that is fine. Precision matters more than polish.
